Good first additions: the museum kiosk + coral-simulator shots, the curved DataXplorer screen shot, the round touch-table shot, and the at-home desktop game shot (the reference photos this skill's style is based on). Name by setting + topic, e.g.:

```text

02-exhibition-curved-screen-dataxplorer.png
```
