# Style-calibration examples

Drop a few on-style storyboards here as **visual calibration only** — to anchor line weight, palette restraint, whitespace, and how the stick-figure actor participates.

Good first additions: the OceanX coral-recognition strip and the marine-life timeline strip (the reference images this skill's style is based on). Name them by topic, e.g.:

```text
01-coral-recognition-strip.png
02-marine-timeline-shift.png
```

These are **not** templates. Do not copy their panel compositions, props, or labels — reinvent the staging for each new experience (see `../references/storyboard-patterns.md`, "Freshness rule").
